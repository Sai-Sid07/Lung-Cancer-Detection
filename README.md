# 19CSE431 - Digital Image Processing
## Project Topic: Lung Cancer Detection
## Description: 
To take various X-Ray Images with varying features like Over-Exposed Images, Dim/Dull Images, Low Contrast Images, High Contrast images, Images with varying levels and varying types of noise. Various operations are done on these images and outputs are seen and noted and appropriate filters which further enhance the image are shortlisted based on their outputs and relevance.

### Implementation:
  * Basic Image operations
  * Intensity transformation
  * Spatial Filtering
